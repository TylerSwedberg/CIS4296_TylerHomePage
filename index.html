Maze Solver

During my time taking java here at Temple, one of my most memorable project I have done is the maze solver. As a kid I would always try and solve mazes whether in a book or in a magazine. I always wondered how people think when they make these mazes as well as how do people make it complex enough for others to solve. Being able to create a maze using the program given I was able to come up with an easy solution by using stacks. When a program creates a specific dimension of the maze, it will be able to understand how it can reach the end using the stack. (pretty much taking paths until it reaches the end.)
